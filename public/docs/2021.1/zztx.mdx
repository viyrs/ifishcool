在学校，每天了解最新新闻是政治课的一项作业，也是紧跟时代的一种方式。班主任要求每晚自习前观看半小时新闻，我们通常播放的是江西卫视的《杂志天下》。

![](./docs/assets/images/QQ20260121-173456.png)

常规的做法是：打开江西卫视网站，找到最新一期视频，点击播放。但每次播放前，都有一分钟左右的广告——日复一日，不仅浪费时间，操作也显得重复而低效。

作为班里的电教员，我一直在想：有没有更快捷、更省事的方式？

既然它是个网页，视频资源应该可以直接获取。通过查看网页源代码，果然轻松找到了视频的 mp4 直链。

![](./docs/assets/images/QQ20260121-173841.png)

测试一下，直接打开这个链接就能播放，**没有广告**，也不需要等待。

目标明确了：能不能写一个程序，每天自动获取最新视频，并用系统播放器直接打开？

于是我用 Python 写了一个脚本：它先爬取江西卫视的页面，解析出最新一期的视频链接，再通过调用系统命令，直接用 Windows 自带的播放器打开。

![](./docs/assets/images/QQ20260121-173721.png)

效果出乎意料地好——不仅广告没了，整个操作也从“打开网页-等待-点击”简化为“双击程序”。

后来，我还为它设置了定时任务：每天晚上 7 点半自动运行脚本，播放新闻。从那以后，我再也不用提前到教室手动操作，全班同学也能准时、无打扰地开始收看。

一个小小的脚本，让一件重复的日常变得轻盈。技术不一定非要宏大，能解决身边的具体问题，就已足够美好。
