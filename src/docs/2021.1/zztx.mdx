在学校，每天了解最新新闻是政治课的一项作业，也是紧跟时代的一种方式。班主任要求每晚自习前观看半小时新闻，我们通常播放的是江西卫视的《杂志天下》。

![](https://cdn-community.bcmcdn.com/47/community/d2ViXzMwMDFfODQzODE4XzBfMTc2OTAwMzA0NDg4MV85M2E1ZGNlNg.png)

常规的做法是：打开江西卫视网站，找到最新一期视频，点击播放。但每次播放前，都有一分钟左右的广告——日复一日，不仅浪费时间，操作也显得重复而低效。

作为班里的电教员，我一直在想：有没有更快捷、更省事的方式？

## 找到突破口：直接获取视频源

既然它是个网页，视频资源应该可以直接获取。通过查看网页源代码，果然轻松找到了视频的 mp4 直链。

测试一下，直接打开这个链接就能播放，**没有广告**，也不需要等待。

目标明确了：能不能写一个程序，每天自动获取最新视频，并用系统播放器直接打开？

## 实现方案：用 Python 把流程自动化

于是我用 Python 写了一个脚本：它先爬取江西卫视的页面，解析出最新一期的视频链接，再通过调用系统命令，直接用 Windows 自带的播放器打开。

![](https://cdn-community.bcmcdn.com/47/community/d2ViXzMwMDFfODQzODE4XzBfMTc2OTAwMzA2MDMxNF82OTBkYjM2Yw.png)

真正写脚本的时候，也踩过不少坑。

一开始我只是在浏览器里按 F12，随便搜了一下 `mp4`，就把链接复制出来了。

但很快我发现：有时候网页结构会改，或者视频地址会带上时间戳、签名参数，**今天能用的规则，明天可能就失效**。

于是我重新整理了整个思路：

1. 先用 `requests` 把页面 HTML 抓下来
2. 用正则或解析库定位出「最新一期」的那块区域
3. 再从这块区域里提取出真正的视频地址
4. 最后才是调用系统命令播放

这样做的好处是：  
就算网站的布局稍微有调整，只要核心结构不变，脚本也不用每天跟着改。而且把“找视频”和“播放视频”拆开之后，后面要换播放器、或者改成只下载不播放，都变得很容易。

## 边界与规则：技术之外的自我约束

在实践中，我也开始更注意**边界和规则**：  
比如尽量控制访问频率，只在每天固定时间请求一次页面；  
只做“正常用户自己会手动完成的事情”的自动化，而不是去爬取、传播任何付费或敏感内容。  
这是一个「尊重服务方、尊重版权」的态度问题。

## 收获：一个小脚本带来的长期影响

这段小项目还有一个意外收获：  
之前同学们对“编程”的印象大多停留在刷题、做练习上，而当他们看到“一个几十行的 Python 脚本就能帮全班省下每天的广告时间”时，会突然明白——

> 原来写代码，真的可以直接把不爽的日常，改成更顺手的版本。

效果出乎意料地好——不仅广告没了，整个操作也从“打开网页-等待-点击”简化为“双击程序”。

### 脚本的最后一步：自动化播放

后来，我还为它设置了定时任务：每天晚上 7 点半自动运行脚本，播放新闻。从那以后，我再也不用提前到教室手动操作，全班同学也能准时、无打扰地开始收看。

### 小结：技术的价值在于解决实际问题

一个小小的脚本，让一件重复的日常变得轻盈。技术不一定非要宏大，能解决身边的具体问题，就已足够美好。
