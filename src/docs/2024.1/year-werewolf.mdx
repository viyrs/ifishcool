在今年春节快到之时，我想到了一个特有意思的玩法。照理说，除夕夜本是家庭和睦的一晚，大家一起守岁、期待新的一年，是非常「团圆」「温暖」的场景。

但是如果除夕夜和狼人杀结合起来呢？

- 跨年失败？
- 死在除夕？

这种巨大的反差本身就很有画面感，再加上春节的烟花氛围、稍微恐怖一点的场景氛围，让整个跨年夜既有仪式感，又充满刺激和惊喜。

当时我的想法很简单：**想让玩家在一个非常熟悉的节日节点，体验到完全不一样的紧张感和代入感。**

在给这张图起名字的时候，我想到了「除夕惊魂夜」这四个字：

- 「除夕」代表的是那种大家默认的团圆、温暖、安全感；
- 「惊魂夜」则是狼人杀带来的不确定、怀疑和紧张气氛。

这两个词放在一起，就有很强的反差和画面感，也刚好概括了这张地图想给玩家的体验。

## 策划玩法和游戏制作：一个人把全流程兜住

说干就干，我先开始策划玩法细节和整体 2D UI。因为这次项目从策划到实现几乎都是我一个人包揽，所以一开始就在想：

- 玩法要足够简单、易上手，适合朋友一起开黑；
- UI 要一眼看出「春节」和「狼人杀」两个关键词；
- 整体流程要能撑起一整局跨年的紧张感。

## 核心玩法：经典狼人杀规则 + 节日氛围加成

整体的核心玩法我没有刻意做得太花，而是保留了狼人杀本身的乐趣，再叠加春节场景的氛围：

- 狼人杀玩法本身不变；
- 根据每回合人数，自动分配不同数量的狼人、猎人和村民；
- 所有人都被投放到一个有强烈节日氛围的场景中；
- 一局坚持 15 分钟，最后看哪一方胜出。

这种设计的好处是：

- 老玩家几乎不用学规则，上来就能玩；
- 新玩家只要跟着队友节奏，很快就能理解谁是「好人局」、谁在「搞事」；
- 氛围主要靠场景和 UI 来给，规则本身保持相对克制。

## 小玩法设计：任务、金币、瞬移和烟花

为了让每一局都不只是「说话投票」，我在地图里加了一些小玩法，让整局游戏更有节奏感和随机事件：

- **紧急任务**：在关键时刻抛出一个必须处理的目标，打乱大家原本的节奏；
- **收集金币**：在场景中穿插资源点，让玩家在游走时有「顺手拿点东西」的反馈；
- **瞬移**：让玩家可以快速位移、调整站位，也会制造很多「刚才他明明在那边」的讨论点；
- **放烟花**：一方面是春节氛围必备，另一方面在紧张局面里突然来一发烟花，本身就很有反差感，也算一个小小的情绪缓冲。

这些小玩法本质上都是在给玩家增加 buff，让每局游戏不只是「投票谁出局」，而是有更多临场决策和临时变数。

## 实际体验：16 人一局的跨年狂欢

这个作品最多支持 16 人一局一起玩，人多的时候现场真的非常热闹：

- 不同身份的人在同一个春节小镇里各怀心思；
- 有人在认真分析，有人在瞎搅和，有人在默默完成任务；
- 配合烟花、灯光和节日配色，聊天频道的气氛也会被带起来。

那种「一群人在线上，却过出了线下跨年局的味道」的感觉，是我当时最想做出来的东西。
